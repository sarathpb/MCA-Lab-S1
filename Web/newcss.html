This page will help you understand how CSS works using all 3 types: Inline, Internal, and External CSS.

CSS Syntax

CSS rules are made up of selectors and declaration blocks.

selector {
  property: value;
}

Types of CSS

  • Inline CSS – applied directly to an element via the style attribute
  • Internal CSS – defined inside <style> tag in the HTML document
  • External CSS – written in a separate file (like styles.css)

Element Selector

The element selector selects elements based on the tag name. For example:

p {
  color: blue;
}

It will make all <p> tags blue.
